# HR_Analytics_Dashboard

## Project Overview
This project analyzes HR data to understand employee attrition and workforce trends using Power BI. The dashboard provides interactive visualizations to identify key factors affecting employee turnover and helps organizations make data-driven HR decisions.

## Tools Used
* Power BI
* Excel / CSV Dataset
* Data Visualization

## Key Features
* KPI cards for Total Employees, Attrition, Attrition Rate, Average Age, Average Salary, and Years at Company
* Attrition analysis by Education Field
* Attrition analysis by Age Group
* Attrition analysis by Job Role
* Attrition analysis by Salary Slab
* Attrition analysis by Years at Company
* Attrition analysis by Gender and Department filters

## Key Insights
* Most employee attrition occurs in the **26–35 age group**.
* Employees with **salary up to 5k** show the highest attrition rate.
* **Laboratory Technicians and Sales Executives** have higher attrition compared to other roles.
* Majority of employees belong to the **Life Sciences and Medical education fields**.

## Conclusion
This dashboard helps HR teams identify patterns behind employee attrition and supports better workforce planning and retention strategies.
